MovieMatch

Project created using ReactNative, Git, Github, Expo, js, and the Movie DataBase's API. This project was created to help users find movies to watch based on their preferences. The app is divided into two main sections, one for movies and one for series, both work by displaying the genres available for each and making an API call for movies related to that genre.

TempTrack

Created a dynamic mobile app that displays real-time weather conditions based on user's location. Integrated OpenWeatherMap's API for weather condition data fetching. Implemented expo location a library that provides us with geolocation data, enabling the app to fetch the data of one of the 200,000 cities listed on OpenWeatherMap.

Report Generator

Engineered a report generator using Python and the Google Calendar's API. This project was implemented using the openpyxl library for creating excel reports, and the BeautifulSoup library for cleaning the API results. This resulted in the automation of the process of creating reports for the company, saving the company time and money.
