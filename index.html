Suggestify

Suggestify is an innovative music recommendation app created with Next.js, NextAuth, MongoDB, and Spotify's API to help users find music tailored to their preferences. The app features two main sections: one for music suggestions based on user searches and another that offers recommendations from a list of the user's top tracks. Users can effortlessly save these suggestions to their Spotify playlists, making music discovery and management both intuitive and enjoyable.

MovieMatch

MovieMatch is an intuitive movie and series recommendation app developed with React Native, Git, GitHub, Expo, JavaScript, and The Movie Database (TMDb) API. Designed to help users discover new movies and series based on their genre preferences, MovieMatch features two main sections: one for movies and another for series. By displaying available genres and making API calls for related content, the app provides a seamless and engaging experience for users to find their next favorite show or film.

TempTrack

TempTrack is a dynamic mobile weather app designed to display real-time weather conditions based on the user's location. Leveraging OpenWeatherMap's API and Expo Location, TempTrack fetches accurate weather data for any of the 200,000 cities listed on OpenWeatherMap. With a responsive and user-friendly interface, TempTrack ensures you are always informed about the current weather conditions, no matter where you are.

Report Generator

The Report Generator is a powerful Python-based tool designed to automate the creation of reports using the Google Calendar API. Leveraging the openpyxl library for generating Excel reports and BeautifulSoup for data cleaning, this project automates the report creation process, saving your company valuable time and resources. With a simple and intuitive UI crafted using Tkinter, it makes report generation easy and efficient, ensuring accurate data handling and enhancing productivity.
